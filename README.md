AMP Validator
===

A command line tool for validating AMP HTML pages.

Usage
---

```
npm install -g amp-validator
amp-validator https://www.theguardian.com/sport/2015/nov/10/russia-iaaf-facing-suspension-wada-doping-report/amp
```
